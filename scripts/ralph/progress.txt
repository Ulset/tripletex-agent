## Codebase Patterns
- OpenAI is used via `from openai import OpenAI` with chat.completions.create()
- Tests mock OpenAI at the class level with `@patch("src.agent.OpenAI")`
- TripletexClient methods: get(endpoint, params), post(endpoint, json), put(endpoint, json), delete(endpoint)
- TripletexAPIError(status_code, message) is raised on 4xx/5xx responses
- Knowledge base is imported via `from src.knowledge import TRIPLETEX_API_REFERENCE`
- Use `python3 -m pytest tests/ -v --ignore=tests/e2e` to run all non-e2e tests
